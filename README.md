# SUNSTAR Modbus RTU 控制器監控軟體 (V3.0)

## 簡介
本程式專為 SUNSTAR SY-DPCA 系列雙比例線圈控制器設計，提供直觀的圖形化介面，讓使用者能夠透過 Modbus RTU 通訊協定（RS-485 介面）輕鬆讀取與寫入控制器的各項參數。

## 功能特色
*   **雙模式支援：** 支援 SY-DPCA-*-2（雙控制器）與 SY-DPCA-*-1（單控制器）型號切換。
*   **快速設定精靈 (New!)：** 引導式步驟，協助使用者快速完成基本設定。
*   **即時監控：** 可同時監控兩組輸出的電流、輸入信號及故障狀態。
*   **參數調整：** 提供完整的參數寫入功能，包含最大/最小電流、斜率時間、PWM 頻率、震顫參數及 PID 設定。
*   **圖表分析：** 提供即時數據圖表，視覺化調整輸出曲線特性。
*   **批量操作：** 支援參數的本地儲存、讀取及一鍵寫入控制器。
*   **多語言介面：** 支援繁體中文與英文切換。

## 安裝與使用
本程式為免安裝版本，解壓縮後即可使用。

1.  進入 `SUNSTAR_dual_controller_app` 資料夾。
2.  找到並執行 `SUNSTAR_dual_controller_app_V3.0.exe`。
    *   **注意：** 為了加快開啟速度，程式採用資料夾形式發布。請確保整個資料夾的完整性，不要單獨移動 `.exe` 檔。

### 疑難排解：SmartScreen 警告
在 Windows 系統上首次執行時，可能會出現藍色的 **Windows SmartScreen** 警告視窗，提示「Windows 已保護您的電腦」。
這是因為本程式為內部開發工具，未購買昂貴的微軟數位簽章，屬於正常現象。

**如何繞過警告：**
1.  點擊警告視窗中的 **「其他資訊」 (More info)**。
2.  點擊下方的 **「仍要執行」 (Run anyway)** 按鈕。
3.  之後在同一台電腦上執行將不再出現此警告。

## 系統需求
*   Windows 10 / 11 作業系統
*   USB 轉 RS-485 轉換器（需安裝相應驅動程式）

## 操作流程
1.  **快速設定 (建議)**：
    -   程式會自動開啟「快速設定精靈」。
    -   依照畫面指示，逐步選擇您的應用場景與基本需求。
    -   精靈結束後會自動設定好相關參數。
    -   若不需要快速設定，則在選擇語言與型號後，點選進入主程式按鈕即可直接開啟主程式。
2.  **連接控制器**：
    -   在主程式左上角的「Modbus通訊參數設置」區域選擇正確的通訊端口 (COM Port)、鮑率 (Baud Rate) 和設備位址 (Address)。
    -   點擊「連接」按鈕。
3.  **監控控制器狀態**：
    -   連接成功後，程式會自動開始輪詢讀取控制器狀態，在主程式上方「即時監控區」顯示控制器的狀態，包括電流、輸入信號、故障狀態等。
    -   點擊即時監控區右上方的「即時圖表」按鈕，開啟新視窗觀察電流與信號的即時變化。
4.  **設定參數**：
    -   主程式中間「參數設定區」可進行控制器的參數設定，包括最大/最小電流、斜率時間、PWM 頻率、震顫參數及 PID 設定。
    -   對照主程式的下方靜態特性曲線，確認參數設定是否符合預期。
    -   參數設定完成後，點擊「套用至控制器」按鈕，將所有修改後的參數一次性寫入控制器。
5.  **參數儲存與讀取**：
    -   使用「本地儲存」將調校好的參數保存起來。
    -   使用「本地讀取」載入之前保存的參數方案。
6.  **變更型號或語言 (若有需要)**：在主程式右上角下拉選單中可進行型號和語言的切換。
